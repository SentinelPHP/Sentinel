## [1.0.0] - 2026-07-03

### Changed
- License changed from MIT to GPL v3 (2026-04-27)

### Added
- Initial release extracted from SentinelPHP
- `Generator` for creating PHP DTOs from JSON Schema
- `TypeMapper` for JSON Schema to PHP type conversion
- `ClassBuilder` for PSR-12 compliant code generation
- Naming strategy support
- Nested object and enum generation
