description: Review the application
---

Review the application in its current state.

Ensure that at least these areas are covered:
- Code quality
- Security, this application must be PCI-compliant.
- Performance
- Documentation
- Testing
  - verify that coverage is good enough, but sensible.
  - Doesn't need to test every possible edge case.
  - Also ensure that tests extends the best suited testing class (Unit, Kernel, Browser)
  - Remove redundant tests
- SOLID principle is followed
- Maintainability
- Find any code, annotations and configuration that does not add any value
- Are we using modern PHP features?
